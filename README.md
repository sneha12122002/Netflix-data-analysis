# Netflix Movies Data Analysis

# Project Description

This project involves a comprehensive analysis of a Netflix Movies dataset using Python. The primary objective was to clean, transform, and analyze movie-related data to uncover meaningful insights about genres, popularity, ratings, and release trends.

The project demonstrates the complete data analysis workflow, including data preprocessing, exploratory data analysis (EDA),and data visualization. Various business-oriented questions were explored to better understand audience preferences and movie characteristics.

# Business Problem

Netflix hosts thousands of movies across different genres and time periods. Understanding which genres perform well, how ratings are distributed, and how movie releases have changed over time can help stakeholders make informed content decisions.
This analysis aims to identify patterns and trends within the dataset through data-driven exploration.

## 📂 Dataset Overview

The dataset contains movie-related information such as:

* Movie Title
* Genre
* Popularity Score
* Vote Average
* Vote Count
* Release Date
* Language
* Overview

# Project Workflow

# 1. Data Collection

* Imported the Netflix movies dataset into Python using Pandas.

# 2. Data Understanding

* Examined dataset structure.
* Checked data types and summary statistics.
* Explored dataset dimensions and column information.

# 3. Data Cleaning

* Identified and handled missing values.
* Removed unnecessary columns.
* Checked and removed duplicate records.
* Converted date columns into proper datetime format.

# 4. Feature Engineering

* Extracted release year from release date.
* Categorized movie ratings into meaningful groups.
* Split multiple genres into individual records for detailed analysis.

# 5. Exploratory Data Analysis (EDA)

Performed analysis to answer questions such as:

* Which genres are most common?
* Which movies are the most popular?
* How are ratings distributed?
* Which years had the highest movie releases?
* What trends exist across genres and ratings?

# 6. Data Visualization

Created charts and visualizations to better understand:

* Genre distribution
* Popularity trends
* Rating patterns
* Release year trends

# Key Insights

* Drama and Comedy genres were among the most frequently occurring categories.
* Most movies fell into moderate rating ranges.
* A small number of movies accounted for exceptionally high popularity scores.
* Movie releases showed growth across several years.
* Genre-wise analysis revealed noticeable differences in audience engagement.


# Project Outcome

This project transformed raw movie data into meaningful insights through systematic analysis and visualization. It highlights practical data analytics skills commonly used in real-world business environments and demonstrates proficiency in Python-based data analysis.
